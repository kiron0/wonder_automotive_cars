## Installation

```bash
git clone https://github.com/ProgrammingHeroWC4/warehouse-management-client-side-kiron0.git
```

```bash
cd warehouse-management-client-side-kiron0
```

```bash
npm install
```

```bash
code .
```

```bash
npm start
```
----
